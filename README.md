# 📱 Projeto 1 - Mobile

Este projeto foi desenvolvido utilizando o framework Ionic com Angular, com o objetivo de criar uma aplicação mobile simples contendo abas e um botão de alerta.

---

## 🚀 Funcionalidades

* Aplicação com navegação em abas (tabs)
* Botão centralizado na tela
* Exibição de alerta ao clicar no botão
* Interface simples e intuitiva

---

## 🛠️ Tecnologias utilizadas

* Ionic
* Angular
* TypeScript
* HTML e CSS

---

## 📋 Descrição

A aplicação possui três abas:

* **Senhas**
* **Relatórios**
* **Sobre**

Na primeira aba, há um botão chamado **"Alerta"**.
Ao clicar nele, é exibida uma mensagem contendo:

> Nome Sobrenome - Matrícula

---

## ▶️ Como executar o projeto

1. Instale as dependências:

```
npm install
```

2. Execute o projeto:

```
ionic serve
```
